# Velonotes API!

This API supports http://velonotes.com

https://github.com/toddcd/velonotes-client

## Route Endpoints
1. '/api/register'
    * POST '/register'

2. '/api/auth'
    * POST '/login'  
      
3. '/api/bicycles'
    * GET '/'
    * POST '/'  
    * GET '/:bike_id'  
        
4. '/api/positions'  
    * POST '/'  
    * GET '/:position_id'  
    * DELETE '/:position_id'  
    * PATCH '/:position_id'  

5. '/api/notes'  
    * POST '/'  
    * GET '/:note_id'  
    * DELETE '/:note_id'  
    * PATCH '/:note_id'  
    
6. '/api/uidata'  
    * GET '/make'  
    * GET '/size'
